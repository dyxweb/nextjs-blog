## 接口请求错误数据采集
> 接口请求错误一般可分为两种，一种是http状态码错误类型比如接口请求404，还有一种是业务逻辑错误，http状态码正常，但是根据约定返回的响应数据格式判断为业务逻辑错误。

### 接口请求的http错误上报
```
export const httpErrorMonitor = error => {
  const {
    config: { url, method, data, params },
    response: { status },
    message
  } = error;
  lazyReportCache({
    type: 'requestError',
    subType: 'http',
    time: dayjs().format('YYYY-MM-DD HH:mm:ss'),
    page: location.href,
    data: JSON.stringify({
      path: url,
      method,
      status,
      requestData: method === 'get' ? params : (data ? JSON.parse(data) : null),
      message
    })
  });
};
```
### 接口请求的业务错误上报
```
export const businessErrorMonitor = error => {
  const {
    config: { url, method, data, params },
    status,
    data: { code, msg }
  } = error;
  lazyReportCache({
    type: 'requestError',
    subType: 'business',
    time: dayjs().format('YYYY-MM-DD HH:mm:ss'),
    page: location.href,
    data: JSON.stringify({
      path: url,
      method,
      status,
      requestData: method === 'get' ? params : (data ? JSON.parse(data) : null),
      message: msg,
      businessCode: code,
    })
  });
};
```
### 以axios为例上报请求错误
```
import axios from 'axios';
import { message } from 'antd';
import qs from 'qs';

const request = axios.create({
  baseURL: '',
  timeout: 300000,
  paramsSerializer: function (params) {
    return qs.stringify(params, { arrayFormat: 'brackets' });
  }
});

request.interceptors.response.use(
  response => {
    const {
      data: { code, msg }
    } = response;
    if (code && code !== 0) {
      message.warning(msg);
      // code非0表示接口业务逻辑错误
      businessErrorMonitor(response);
    }
    return response;
  },
  error => {
    message.error(error.message || '请求失败，请重试');
    // 接口请求http错误
    httpErrorMonitor(error);
    return Promise.reject(error);
  }
);

export default request;
```
